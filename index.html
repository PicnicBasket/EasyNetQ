EasyNetQ

An easy .NET API for RabbitMQ. Documentation

EasyNetQ's aim is to provide developers with a very simple API that abstracts away the Exchange/Binding/Queue model of AMQP and instead provides a simple publish/subscribe and request/response model. Messages are .NET types and routing is by message type.
